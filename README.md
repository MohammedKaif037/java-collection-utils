# Java Collection Utils

**Task 1:** `filter()` method transforms each element in an array using a custom rule and returns a new array.

**Task 2:** `countElements()` method counts how many times each element appears in an array and returns a Map.

**How to run:** Compile with `javac CollectionUtils.java Main.java` then run with `java Main`

**Supports:** Any data type (String, Integer, Character, etc.)
